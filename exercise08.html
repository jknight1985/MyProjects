jQuery - JavaScript library

JavaScript is the most widely-used language for adding dynamic behavior to web pages. The JavaScript community contributes to a collection of libraries that extend and ease its use. In this course, you will learn about jQuery, a JavaScript library that makes it easy to add dynamic behavior to HTML elements.

Example login functionality

const login = document.getElementById('login');
const loginMenu = document.getElementById('loginMenu');

login.addEventListener('click', () => {
  if(loginMenu.style.display === 'none'){
    loginMenu.style.display = 'inline';
  } else {
    loginMenu.style.display = 'none';
  }
});

jQuery makes your life simpler:

$('#login').click(() => {
  $('#loginMenu').toggle()
});

jQuery methods

The pre-made Lego structures are like jQuery methods. You can use these methods to add dynamic behavior, such as .hide(), .show(), .fadeIn(), .fadeOut() etc., to HTML elements.

It has become common practice to link the main JavaScript file at the bottom of the HTML document because a good deal of the content of the script will require that the dependencies, style sheets and elements exist before the browser can run the JavaScript that uses and references those things.

How to add jQuery

You can load jQuery using the respective CDN.

.ready()

a web page must be rendered in a user's browser before it's possible to have any dynamic behavior. To solve this problem, we will use our first jQuery method .ready().

When a web page is loaded, the browser creates a Document Object Model DOM of the page.

DOM

You should wrap all JavaScript behavior inside of the .ready() method. This will make sure the web page is rendered in the browser before any jQuery code executes in main.js. 

$(document).ready(() => {      
            
});

Document is the query object in this case. Typically, string, class, tag or ID is used. Here is a class example:

$('.someClass').handlerMethod();

Developers often save jQuery objects in variables, like so:

const $jQueryObject = $('.someClass');

It's best practice to start jQuery variables with an $-sign.

Event Handlers - .on()

The jQuery .on() method adds event handlers to jQuery objects. The method takes two parameters: a string declaring the event to listen for (the handler) and a callback function to fire when the event is detected.

$('#login').on('click', () => {
  $loginForm.show();
})

jQuery effects

Use the .toggle function to combine .show and .hide

  $('.toggle-button').on('click', () => {
    $('.first-image').toggle();
  });

With fadeIn() and fadeOut() you can hide and show with a time variable for effect.

 $('.fade-out-button').on('click', () => {
    $('.fade-image').fadeOut(500);
  });

Review

To review, the methods that we learned were:

  • .hide()
  • .show()
  • .toggle()

These methods instantly hide or show elements on a web page.

  • .fadeOut()
  • .fadeIn()
  • .fadeToggle()

These methods make elements disappear or appear over a given period of time.

  • .slideUp()
  • .slideDown()
  • .slideToggle()
