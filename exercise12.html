Solidity

Solidity is a programming language to programmes the Ethereum Virtual Machine (EVM) to execute smart contracts. This documentation is a "get started" tutorial on smart contracts.

Tools

I will use a bunch of different tools to execute the smart contract.

Metamask

One of which is Metamask. Metamask is a Chrome browser extension that enables the creation of ethereum wallets. Install it by clicking, settings -> add-ons in your Chrome browser. Set your password, save your seed! and start to create some wallets, e. g. in Kovan, one of the test systems of Ethereum. Side note, Kovan is a metro station in Singapore.

MetaMask

In order to execute your smart contract, you need to buy Ether - which as you may have heard - served as a currency for Gas which drives the EVM. There are so called faucets for the test systems Ether, so let's get some.

MetaMask: buy test ether

Clicking buy will redirect you to Github, where different ways to acquire test Ethers are explained.
