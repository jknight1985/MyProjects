K-nearest neighbors (KNN) algorithm

  • does not require training
  • new case is classified based on proximity to labeled cases
  • use a distance metric
  • euclidian distance
  • weighted distance
  • K defines the number of neighbors considered
  • majority vote

Example: How to classify this question mark?

Example: K-nearest neighbors
  • Option 1: K = 1, NN --> only nearest neighbor considered
  • Option 2: K = 3, NN --> three nearest neighbor considered

KNN is an effective classifier, but KNN suffers from the curse of dimensionality. If you more dimensions and you want the same density of data, the samples necessary grows with the power of the dimensions:

  • 1 dimension = 10 samples
  • 2 dimension = 100 samples
  • 3 dimension = 1000 samples
  • 4 dimension = 10000 samples

Affects all ML algorithms, particularly KNN.
