What is a blockchain?

The blockchain is just an accurate, and permanent record of all the transactions that have happened amongst the members in that blockchain’s network.

  1. Block: A block is an individual transaction or piece of data that is being stored within the blockchain.
  2. Blockchain: A blockchain is a continuously growing list ("chain") of records ("block"), called blocks, which are linked chronologically and secured using cryptography.

Blocks

What are blocks?
  1. Timestamp: The time the block is created determines the location of it on the blockchain.
  2. Transaction Data: The information to be securely stored in the block.
  3. Hash: A unique code produced by combining all the contents within the block itself — also known as a digital fingerprint.
  4. Previous Hash: Each block has a reference to the block prior to its hash. This is what makes the blockchain unique because this link will be broken if a block is tampered with.

Hashing

Blockchain uses a cryptographic hash function, meaning that the output is random but deterministic.

  1. Deterministic: The same input will always produce the same output, but that output cannot produce the original input.
  2. Hash: A calculated string of letters and numbers produced from a specific input.
  3. Hash Function: A function that takes in an input of a random size, performs hashing on the input, and generates a seemingly random output of a fixed size, also known as the hash.
