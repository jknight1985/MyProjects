Learning python

Again, I'm building on the good experience, I had with codecademy now learning python. As python also is a script language gaining a lot of popularity in data science, I expect similarities to JavaScript. Let's start!

Python syntax

As in JS you need variables can be assigned without a data type.

my_int = 7

There is no ending semicolon. Intentation however plays quite a significant role! Not doing it will result in errors!

def spam():
  eggs = 12
  return eggs
        
print spam()

Comments are indicated with the # sign. Triple quotation marks are for multiline comments.

"""asdf"""

Math operators are as you would expect, +-*/. ** will give you the exponent.

eggs = 10**2

The modulo % will give you the remainder of a division.

Strings & console output

Apostrophes need to be accompanied by a \ in order to interpret the string correctly.

'This isn\'t flying, this is falling with style!'

You can use the index [] to access string characters. The index starts at 0. The following statement will therefore return Y.

fifth_letter = "MONTY"[4]

String methods

Methods that use dot notation only work with strings. E. g. 

"HELLO".lower()

Methods like len() can work with any data type.

Strings can be concatenated with the + operator.

print "Life " + "of " + "Brian"

The % operator after a string is used to combine a string with variables. %s is replaced with the variable after the %.

print "Let's not go to %s. 'Tis a silly %s." % (string_1, string_2)

Date & time

Use the datetime library to access the current date.

from datetime import datetime
now = datetime.now()

print '%s/%s/%s %s:%s:%s' % (now.month, now.day, now.year, now.hour, now.minute, now.second)

Conditional flow control

You can use comparators:

3 < 4
5 >= 5
10 == 10
12 != 13

Boolean comparators

True or False 
(3 < 4) and (5 >= 5)
this() and not that()

Conditional statements:

if this_might_be_true():
  print "This really is true."
elif that_might_be_true():
  print "That is true."
else:
  print "None of the above."
