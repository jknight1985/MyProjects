Learning python

Again, I'm building on the good experience, I had with codecademy now learning python. As python also is a script language gaining a lot of popularity in data science, I expect similarities to JavaScript. Let's start!

Python syntax

As in JS you need variables can be assigned without a data type.

my_int = 7

There is no ending semicolon. Intentation however plays quite a significant role! Not doing it will result in errors!

def spam():
  eggs = 12
  return eggs
        
print spam()

Comments are indicated with the # sign. Triple quotation marks are for multiline comments.

"""asdf"""

Math operators are as you would expect, +-*/. ** will give you the exponent.

eggs = 10**2

The modulo % will give you the remainder of a division.

Strings & console output

Apostrophes need to be accompanied by a \ in order to interpret the string correctly.

'This isn\'t flying, this is falling with style!'

You can use the index [] to access string characters. The index starts at 0. The following statement will therefore return Y.

fifth_letter = "MONTY"[4]

String methods

Methods that use dot notation only work with strings. E. g. 

"HELLO".lower()

Methods like len() can work with any data type.

Strings can be concatenated with the + operator.

print "Life " + "of " + "Brian"

The % operator after a string is used to combine a string with variables. %s is replaced with the variable after the %.

print "Let's not go to %s. 'Tis a silly %s." % (string_1, string_2)

Date & time

Use the datetime library to access the current date.

from datetime import datetime
now = datetime.now()

print '%s/%s/%s %s:%s:%s' % (now.month, now.day, now.year, now.hour, now.minute, now.second)

You can use comparators:

Function

A function has a header, an optional comment and a body.

def spam():	
  	"""Print eggs to the console.""" 
print "Eggs!"

You can import functions using the import command. If you only want to import a certain function, you may use:

from math import sqrt

That way, you can sqrt() instead of math.sqrt. By using universal import for the whole library, you can achieve the same result:

from module import *

This is however dangerous, as might end up using a function from a different model. You can list the function in the module by typing: 

everything = dir(math)

Type outputs the type of the variable, e. g. str, int, etc.

Lists

Using append you can add items to a list.

letters = ['a', 'b', 'c']
letters.append('d')

Use the len function to get back the length of a list.

list_length = len(suitcase) # Set this to the length of suitcase

When addressing only parts of a list, the first indicator is started with, the last one is not taking into consideration. Following expressions will therefore only contain two entries.

last =  suitcase[4:6]

To include all following entries, just leave the end indicator empty:

frog = animals[6:]

Index & insert

By using index() you find the index value of an object in a list. Insert let's you insert an element at a specific position, pushing following elements backwards.

animals = ["aardvark", "badger", "duck", "emu", "fennec fox"]
duck_index =  animals.index("duck")

# Your code here!
animals.insert(duck_index, "cobra")

List can be sorted by appending .sort().

Dictionaries

You can access an entry via its key

residents = {'Puffin' : 104, 'Sloth' : 105, 'Burmese Python' : 106}
print residents['Puffin'] # Prints Puffin's room number

An empty pair of curly braces {} is an empty dictionary, just like an empty pair of [] is an empty list.

You can delete entries by addressing the key.

del zoo_animals['Bengal Tiger']

Or remove by item name

backpack.remove('dagger')

.items() and .keys()

While .items() returns an array of tuples with each tuple consisting of a key/value pair from the dictionary:

  • The .keys() method returns a list of the dictionary's keys, and
  • The .values() method returns a list of the dictionary's values.

Lists & functions

pop let's you remove a the indexed item, while .remove addresses the item by its content. del is like pop.

To address a an element of a list in for loop, use this code

def double_list(x):
  for i in range(0, len(x)):
  	x[i] = x[i]*2
  return x

Range object

The range function has three different versions:

  • range(stop)
  • range(start, stop)
  • range(start, stop, step)

In all cases, the range() function returns a list of numbers from start up to (but not including) stop. Each item increases by step. If omitted, start defaults to 0 and step defaults to 1.

Join

Loops

While loops count until the condition is no longer true.

while count < 10:
  print "Hello, I am a while and count is", count
  count += 1

Break can also be used as a statement to stop a loop.

While/Else

It is also possible to combine while and else. Once your condition is true, the else block will be executed.

count = 0
while count < 3:
  num = random.randint(1, 6)
  print num
  if num == 5:
    print "Sorry, you lose!"
    break
  count += 1
else:
  print "You win!"

After three executed blocks without a 5, the else is executed.

Anonymous function

Short form:

lambda x: x % 3 == 0

Same as: 

def by_three(x):
  return x % 3 == 0

But...function does not need to be defined. Using filter, you can take a second argument to use the anonymous function on:

print filter(lambda x: x % 3 == 0, my_list)
