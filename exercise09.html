Range object

The range function has three different versions:

  • range(stop)
  • range(start, stop)
  • range(start, stop, step)

In all cases, the range() function returns a list of numbers from start up to (but not including) stop. Each item increases by step. If omitted, start defaults to 0 and step defaults to 1.

Final class exercise

The constructor for derivate classes has to contain all arguments of the parent class plus additional.

class Car(object):
  condition = "new"
  def __init__(self, model, color, mpg):
    self.model = model
    self.color = color
    self.mpg   = mpg
   
  def display_car(self):
    print "This is a %s %s with %s MPG." % (self.color, self.model, str(self.mpg))
    
  def drive_car(self):
    self.condition = "used"
    
class ElectricCar(Car):
  def __init__(self, model, color, mpg, battery_type):

    self.color = color
    self.mpg   = mpg

File Input / Output

So far, we just used console input. In this case, the data comes from a file.

Write

my_list = [i ** 2 for i in range(1, 11)]
# Generates a list of squares of the numbers 1 - 10

f = open("output.txt", "w")

for item in my_list:
  f.write(str(item) + "\n")

f.close()

"w" stands for write. This opens the file in write mode, enabling sending data to it. Passing r+ enables read and write, r just read.

.write() let's you write to the file.

Writing won't be done properly, unless you .close() the file in the end. During the I/O process, data is buffered: this means that it is held in a temporary location before being written to the file. Python doesn't flush the buffer, i. e. write data to the file until, it's sure you're done writing.

Read

Read from a file.

my_file = open("output.txt", "r")

print my_file.read()

my_file.close()

You can also read from individual lines. The following code will give you the documents information line by line (automatically counts up!).

print my_file.readline()

With and as

Closing a file can be done automatically by Python. When a file object's __exit__() method is invoked, it automatically closes the file. How do we invoke this method? With with and as.

with open("text.txt", "w") as textfile:
  textfile.write("Success!")

Using the .closed function you can check, whether a file is closed (true or false).
